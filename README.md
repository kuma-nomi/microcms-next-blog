# kumanomi's portfolio

URL:
https://microcms-next-blog-sage.vercel.app/

## 技術構成

- Next.js（13.1.5）
- Typescript（4.9.4）
- microCMS
- Sass
- Tailwind CSS
- ESLint
- Prettier
- Vercel

## 開発方法

### package install

```bash
npm run install

```

### dev

```bash
npm run dev
```

### build

```bash
npm run build

```

## 環境変数

```
SERVICE_DOMAIN=microCMSのサービスID
API_KEY=microCMSのAPIキー
```
